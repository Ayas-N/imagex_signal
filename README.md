# Arduino Peripheral:
1. Connect to an Arduino Nano 33 BLE using USB, through the Arduino IDE, upload the contents of LED.ino to begin running the peripheral.
2. With the Nano 33 in hand, perform various gestures I.e. Front, Back, Left and Right.

# Python BLE Central:
1. Run py central.py, in it's current form it is configured to connect to the Nano's specific UUID using Bluetooth
2. Once connected, the BLE Central will display the direction of motion.
